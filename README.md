# webappsecuritytest
This is UI tests corresponding the manual tests as mentioned in the Google sheet: https://docs.google.com/spreadsheets/d/1qRKuai09mrPUpAAthyLs-JJ7_mQ6qQI8yFR0EBSaudc/edit?usp=sharing

# Clone from repository
Use the following command: <b>git clone https://github.com/askchiranjit/webappsecuritytest.git</b>

# Run the test

Navigate to the directory: <b>/webappsecuritytest</b>
<p>Use the following command: <b>mvn surefire-report:report</b>

# Report
After execution is finished, open the file: <b>surefire-report.html</b> from directory <i>/target/site</i>
